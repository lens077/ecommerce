生效日期：2026 年 03 月 03 日
维护主体：https://github.com/sunmery
项目仓库：https://github.com/lens077/connect-example

特别提示
本平台是一个非商用的开源技术演示项目，旨在展示 B2B2C 电商架构、地址微服务、Casdoor 身份认证以及高德地图 LBS 服务的集成方案。

我们深知个人信息对您的重要性，严格遵守相关法律法规，遵循合法、正当、必要原则。请您知悉：本演示环境会采集部分网络及位置数据，请勿在本平台上输入任何真实的银行卡号、支付密码或家庭详细住址等高度敏感信息。

一、 我们如何收集和使用您的个人信息
我们仅会为实现演示功能（如自动匹配收货地址、快递员派送模拟）收集最小必要范围的信息：

1. 账号演示信息
   认证系统：我们使用 Casdoor 统一身份认证。为完成注册演示，我们会收集您的电子邮箱或第三方社交平台（仅用于验证码模拟）。

演示账号：您可以选择使用我们提供的测试账号登录，此时无需提供您的个人信息。

2. 地址与位置信息（核心演示逻辑）
   IP 地址获取：当您访问演示站点时，系统会记录您的公网 IP 地址。

第三方解析：我们会将您的 IP 地址发送至高德地图 API，以换取对应的行政区划代码（adcode）。

目的说明：此数据仅用于在“新增地址”页面为您自动匹配省、市、区信息，提升用户体验模拟效果。

3. 快递员场景模拟
   地理位置权限：在模拟“快递员实时定位”或“配送路径规划”场景中，经您点击授权后，我们会通过高德 SDK 收集您的经纬度坐标。这些数据仅用于前端地图渲染和派单算法逻辑演示。

二、 第三方服务说明（高德地图 SDK）
为实现地理定位和地图功能，我们集成了高德开放平台服务：

共享数据：IP 地址、经纬度坐标。

数据用途：IP 定位、逆地理编码（坐标转文字地址）、路径规划展示。

隐私参考：详细处理规则请查阅 高德地图隐私政策。

三、 我们如何存储您的个人信息
存储地域：所有演示数据存储于中华人民共和国境内的云服务器。

存储期限：

临时性：作为演示站点，我们将不定期重置数据库（清空所有用户、订单及地址记录）。

无持久化倾向：您的个人信息不会被用于商业分析或精准营销。

技术架构防护：

后端接口采用 JWT 加密鉴权。

数据库（PostgreSQL/Redis）运行在 Docker/K8S 私有子网中，不直接暴露于公网。

全站强制启用 HTTPS (TLS 1.2/1.3)。

四、 开源免责与数据删除
开发者权利：本代码完全开源，您可以自行克隆代码并在本地断网环境下运行，此时我们不会收集任何信息。

数据删除请求：由于是演示环境，如果您曾输入过真实信息并希望删除，请在 GitHub 仓库通过 Issue 告知我们，我们将在 48 小时内从数据库中物理删除相关记录。

五、 联系方式
如果您对本开源项目的隐私保护有任何建议，或发现安全漏洞，请通过GitHub Issue联系
